--- categories: - Informatique date: 2012-10-12T00:00:00Z meta: _edit_last: "1" published: true status: publish tags: - Application Cache - Cache - Chrome title: Cache-cache Chrome type: post --- Ces temps, je suis en plein dans le développement d'une application mobile. De ce fait, je dois m'appuyer sur les différents systèmes de cache à disposition. Chrome est le navigateur que j'utilise pour mes tests. Ne me souvenant jamais des différents raccourcis pour vider les caches, j'ai décidé de les écrire une bonne fois pour toutes.
  • La combinaison de touche CTRL + F5 purge du cache navigateur
  • chrome://appcache-internals/ tapé dans l'url permet de visualiser et supprimer le contenu de "l'application cache".
  • La combinaison de touche CTRL + SHIFT + DEL permet d'effacer les données de navigation.
