+++ categories = ["Internet", "Programmation", "Typo3"] date = "2012-02-10T00:00:00Z" published = true status = "publish" tags = ["Heading", "Mapping", "TemplaVoilà", "Typo3", "TypoScript"] title = "Changer le niveau de titre des éléments de contenu" type = "post" [meta] _edit_last = "1" _syntaxhighlighter_encoded = "1" hide_post_date = "default" hide_post_image = "default" hide_post_meta = "default" hide_post_title = "default" layout = "default" post_color = "default" tmac_last_id = "256453638317281280" unlink_post_title = "default" +++ Dernièrement j'ai eu le cas d'un mapping TemplaVoilà avec Typo3 dans lequel il fallait que les éléments de contenu aient des titres de niveau deux et de niveau trois dans la colonne latérale. Il s'agissait d'un site classique en trois colonnes :
  • 1ère colonne : menu est navigation
  • 2ème colonne : information principale
  • 3ème colonne : information secondaire (<aside>)
Les éléments de la colonne principale devaient avoir des titre de niveau deux (<h2>) et ceux de la colonne de latérale de niveau 3 (<h3>). Lors du mapping TemplaVoilà, il suffit d'ajouter le typoscript suivant sur votre élément dans la structure de donnée (data structure). [code highlight="1,2"] 5 = LOAD_REGISTER 5.defaultHeaderType = 3 10= RECORDS 10.source.current=1 10.tables = tt_content 10.wrap = <!--TYPO3SEARCH_begin--> | <!--TYPO3SEARCH_end--> 15 = RESTORE_REGISTER [/code] A la ligne deux, on définit le niveau du titre. Voilà pour le petit truc ;)
