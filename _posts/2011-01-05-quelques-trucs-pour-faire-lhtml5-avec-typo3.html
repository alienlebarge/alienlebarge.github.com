Par soucis de découverte et d'apprentissage, j'ai opté pour le HTML5. Voici quelques trucs (et autre) que j'ai dû mettre en place pour ce faire :
  1. Lire HTML5 for Webdesigners
  2. Faire un template en respectant la norme et utilisant les nouvelles balises (<nav>, <header>, etc.)
  3. Modifier le doctype dans le setup du gabarit de base. [code] config.doctype = html_5 config.xmlprologue = none [/code]
  4. Utiliser le JavaScript HTML5 IE enabling script pour toutes les versions de IE inferieur à la 9 (merci IE). Il suffit d'ajouter le code suivant dans le setup du gabarit de base [code] #if MS Internet Explorer [browser = msie] AND [version < 9] page.headerData = CARRAY page.headerData { 10 = TEXT 10.value ( <!--[if lt IE 9]> <script src="http://html5shiv.googlecode.com/svn/trunk/html5.js"></script> <![endif]--> ) } [end] [/code]
  5. etc...
Si vous avez d'autre trucs ou configuration, n'hésitez pas à les partager ;)
